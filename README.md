# 💰 Finance App

Aplicação web para controle de contas e organização financeira anual.

Projeto desenvolvido com foco em prática de JavaScript puro (Vanilla JS), manipulação de DOM, autenticação simples e persistência de dados com LocalStorage.

---

## 🚀 Funcionalidades

- 🔐 Sistema de login e cadastro de usuário
- 💾 Persistência de dados por usuário (LocalStorage)
- ➕ Cadastro de contas por mês
- 📊 Visualização organizada em formato de board anual
- 💰 Cálculo automático de total mensal e anual
- ✅ Marcar conta como paga
- 🗑️ Exclusão de contas
- 🔄 Drag and Drop entre meses
- 📱 Layout responsivo

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3 (Flexbox, Glassmorphism UI)
- JavaScript (ES6+)
- LocalStorage API

---

## 🧠 Conceitos Aplicados

- Manipulação de DOM
- Organização modular de código
- Separação de responsabilidades (auth / bills / app)
- Eventos e EventListeners
- Persistência no navegador
- Responsividade com media queries
